* [Twilio](https://twilio.com) - Twilio's WhatsApp messaging API
* [Shopemaa](https://shopemaa.com) - Shopemaa's headless eCommerce API
* [Facebook](https://developers.facebook.com/docs/) - Facebook messaging API & Webhook
